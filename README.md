# URL Checker

Утилита для параллельной проверки доступности списка URL-адресов с измерением времени отклика.

## Возможности

- Параллельная проверка множества URL с настраиваемым количеством воркеров
- Измерение времени отклика (latency) для каждого URL
- Автоматическое добавление префикса `http://` к адресам без протокола
- Детальный отчет с таблицей результатов и статистикой
- Таймаут запросов 10 секунд

## Установка

```bash
go build -o url-checker
```

## Использование

```bash
./url-checker -f sites.txt -w 20
```

### Параметры

- `-f` — путь к файлу со списком URL (по умолчанию: `sites.txt`)
- `-w` — количество параллельных воркеров (по умолчанию: `20`)

### Формат файла с URL

Создайте текстовый файл с URL-адресами (по одному на строку):

```
google.com
https://github.com
example.org
```

## Пример вывода

```
TARGET URL              STATUS  LATENCY     NOTE
---                     ---     ---         ---
http://google.com       200     45ms        OK
https://github.com      200     120ms       OK
http://example.org      ERR     10s         Offline/Timeout

========================================
STATISTICS:
Total Success:  2
Total Failed:   1
Avg Latency:    82ms
Execution Time: 10s
========================================
```
